# Honey landing page dynamic plan.

## Sections to dynamic with fields.

**Hero Section**
- Heading 1 -> text.
- General questions -> text field with add new options.

**Welcome**
- Heading 2 -> text.
- Welcome message -> text.
- Logo -> file upload.

**Description**
- Description -> textarea.

**Why buy our honey**
- Center Image -> image uploader.
- Information cards(5 items):
    - Fields:
        - icon, -> file uploader
        - card heading, -> text
        - card description. -> textarea

**Why eat honey**
- Card with add new options:
    - Card fields:
        - icon, -> file uploader
        - card title, -> text
        - card description. -> textarea

**Review**
- Select options: Screenshot, Review.
- For screenshot option:
    - File uploader
-For Review option:
- Card with add new options:
    - Review card fields:
        - avatar, -> file uploader
        - name, -> text
        - rating, -> stars
        - details -> textarea

**FAQ**
- Card with add new options:
    - Card fields:
        - question, -> text
        - answers -> textarea.

**Product/Price**
- Select options: Existing product, Static product.
- For existing product:
    - Product selection, -> dropdown

- For Static product
    - Product title, -> text
    - Product Image -> file uploader
    - Quantity, -> text
    - Regular price, -> number
    - Offer price, -> number
    - Short description -> textarea/text.
